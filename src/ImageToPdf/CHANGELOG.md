# Changelog

All notable changes to this project are be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] 2023-10-05

Forked from [Algel.ImageToPdf 2.1.0](https://github.com/algel/ImageToPdf).

## Added
- Page size option (`--pagesize`)
- Resolution option (`--resolution`)
- Image width option (`--width`)
- Margin option (`--margin`)

## Changed
- Now only a single file can be specified, and it is an argument instead of an option.
- The tool is now called `img2pdf`.

## Removed
- Removed support for QuestPDF Previewer since it is discontinued for the free license.
